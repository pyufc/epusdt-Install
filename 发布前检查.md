## 检查项

1. 仓库可见性

- 公开仓库可以直接使用 `raw.githubusercontent.com` 一键安装
- 私有仓库不能直接提供公开一键安装入口

2. 上游来源

- 脚本从官方 `GMWalletApp/epusdt` release 下载程序
- 本仓库只维护部署、接管、更新和运维脚本

3. HTTPS 行为

- 部署时填写域名会自动申请 Let's Encrypt 证书
- 域名没有指向当前服务器时会停止安装并提示原因
- 域名模式会自动写入 Nginx 并强制 HTTPS

4. 安装目录

- 推荐使用独立目录：`/www/wwwroot/epusdt`
- 不要使用 `/www/wwwroot`、`/opt`、`/var/www` 这类系统目录或站点根目录

- `README.md`
- `install.sh`
